# TransfermarktParser

---
# Idea
Research and calculate some information from Transfermarkt APIs

## Current features
- Analyse squad for move of their Market Values; 
- Next features can be suggested in the issue tab of repository for future releases.

# Stack
Scrape: selenium  
Parse: bs4, re  
Work with dataframes: pandas

# How to start
1) Download repo  
2) Install required libs from _requirements.txt_  
3) Start _main.py_ script from _src_ directory in your IDE or python console
